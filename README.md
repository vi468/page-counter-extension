# Page Counter

Chrome/Edge extension для подсчёта чего угодно на странице.
Привязка к домену / табу / URL / глобально. Несколько счётчиков одновременно.

## Установка (dev)

1. Открыть `chrome://extensions/`
2. Включить **Developer mode** (верхний правый угол)
3. **Load unpacked** → выбрать папку `page-counter/`
4. Готово. Иконка появится в тулбаре.

После правок кода — жмите 🔄 на карточке расширения в `chrome://extensions/`.

## Использование

- **Клик по иконке** — по умолчанию +1 к главному счётчику страницы. Можно переключить на "открыть попап" в настройках.
- **ПКМ по иконке → Page Counter → попап** — полный интерфейс.
- **Попап**: ★ — главный (его цифра на иконке). Клик по имени — переименовать. Клик по цифре — задать вручную. `⋯` — меню (reset, step, delete, …).
- **Хоткеи**: `Alt+↑` / `Alt+↓` / `Alt+0` (reset). Настраиваются в `chrome://extensions/shortcuts`.

## Scope

| Scope    | Где живёт счётчик                          |
|----------|--------------------------------------------|
| `domain` | Общий для всех страниц домена *(default)*  |
| `tab`    | Только в открытой вкладке                  |
| `url`    | Точный URL (origin + path + query)         |
| `global` | Один на все страницы                       |

## Структура

```
page-counter/
├── manifest.json       # MV3 манифест
├── background.js       # service worker: badge, хоткеи, клик по иконке
├── lib/storage.js      # storage + модель Counter + scope-логика
├── popup/              # UI списка счётчиков
├── options/            # настройки + экспорт/импорт JSON
└── icons/              # 16/48/128 PNG + исходный SVG
```

## Данные

`chrome.storage.local` (локально, без облака). Бэкап — **Настройки → Экспорт JSON**.
